## 🚀 Getting Started  

### 🛠 Requirements  
🔹 A C Compiler (GCC, Clang, or MSVC)  
🔹 A Code Editor (VS Code, Code::Blocks, or Dev-C++)  

### 📌 Compile & Run  

To compile a program, use:  
```bash
gcc filename.c -o output
./output
Or, for Windows:

bash
Copy
Edit
gcc filename.c -o output.exe
output.exe

```
